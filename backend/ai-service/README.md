# AI Service

Inactive FastAPI backend prototype for cloud-powered AI workflows.

This is not the v1 AI backend. The current v1 direction is Firebase Cloud Functions + Genkit in `backend/functions`, with stub flows first and Vertex AI Gemini later. Keep this directory for reference only unless the project intentionally reopens the FastAPI/Cloud Run path.

Do not configure, deploy, or wire iOS to this service for v1.
